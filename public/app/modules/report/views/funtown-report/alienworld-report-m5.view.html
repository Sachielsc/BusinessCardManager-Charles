The installasion and use of gulp watch:

  1. Install Node.js and NPM
    url: https://tecadmin.net/install-latest-nodejs-npm-on-ubuntu/
  2. Install gulp
    npm install gulp -g
  3. In the folder of your project, install local gulp: Npm install
  4. Run “gulp watch” to keep track of your scss file modification
