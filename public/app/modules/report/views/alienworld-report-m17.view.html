I successfully upgrade the node.js on windows:

  1. Detecting if you're vulnerable from the Heroku CLI, i.e. checking whether the current node.js version has vulnerability problems:
    heroku run node -v -a APPNAME
  2. Find out that my current version is old and has a high severity remote Denial of Service (DoS) Constant Hashtable Seeds vulnerability.
  3. Add these code into my package.json to specify the version of Node.js to use on Heroku:
    "engines": {
    "node": "6.11.1"
},
  4. Other useful tips:
    • Heroku Node.js Support:https://devcenter.heroku.com/articles/nodejs-support#specifying-a-node-js-version
    • The new best way to update npm on windows:https://stackoverflow.com/questions/18412129/how-do-i-update-node-and-npm-on-windows
